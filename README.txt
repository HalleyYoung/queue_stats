This program reads a text file which contains two columns of floating point numbers, where the first column is the time something enters a queue and the second column is the service time in queue.  The program prints out the maximum wait time in queue and maximum number of items in the queue.

To install, enter the folder "Queue_stats" through the command prompt and then run "make".  

The program runs on the command line as "queue_stats".   It takes one parameter - the name of the text file.
